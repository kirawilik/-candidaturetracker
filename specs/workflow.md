# Workflow

For every task, the agent must follow strictly:

---

## 1. Analysis
Understand the request clearly and identify requirements.

---

## 2. Specs Review
Read all relevant files in `/specs`:
- project.md
- backend.md
- database.md
- architecture.md
- security.md
- ui.md

---

## 3. Planning
Create a clear step-by-step implementation plan.

---

## 4. Approval (MANDATORY)
Always wait for user approval before implementation.
No execution without confirmation.

---

## 5. Implementation
Execute the approved plan step by step.

---

## 6. Explanation
Explain all changes made in simple terms.
