# UI Rules

## Frontend Stack
- Blade templates (Laravel)

---

## Layout Requirements

- Fully responsive design (mobile, tablet, desktop)
- Clean and minimal dashboard
- User-friendly interface

---

## Components

- Reusable Blade components required
- Avoid code duplication in UI
- Use consistent design patterns

---

## Lists Handling

- Always use @forelse for lists
- Always handle empty states clearly

Example:
- "No data available"
- "No candidatures found"

---
